##  Welcome to the ifcb-analysis wiki!

Here you will find information about analysis tools for Imaging FlowCytobot (IFCB) data. These tools including image annotation, image processing, and image classification routines written in MATLAB. 

##  Requirements:

* MATLAB
* MATLAB Image Processing Toolbox
* MATLAB Neural Network Toolbox
* MATLAB Statistics Toolbox
* MATLAB Curve Fitting Toolbox
* The DIPUM toolbox (which can be purchased [here](http://www.imageprocessingplace.com/DIPUM_Toolbox_2/DIPUM_Toolbox_2.htm))

Plus optional but highly recommended:
* MATLAB Parallel Computing Toolbox
