# 1. Install GitHub Desktop

GitHub Desktop is a GUI for working with GitHub code repositories. You must have a GitHub account to use it. If you don't, see [the instructions for using TortoiseGit](Get-code-using-TortoiseGit-for-Windows).

You can download GitHub here:

https://desktop.github.com/

# 2. Clone the repository

Follow the instructions linked below to clone the repository.

https://help.github.com/desktop/guides/contributing/cloning-a-repository-from-github-desktop/
